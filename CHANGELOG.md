## version 2.0.0 (December 23, 2025)

- **ES6+ syntax** - Leverages modern JavaScript features including arrow functions, template literals, and enhanced object literals

- **New methods** - Added `stop()`, `removeData()`, and `index()` methods
- **Performance improvements** - Optimized animation and DOM manipulation for better performance
- **Bug fixes** - Numerous fixes across width/height methods, data storage, event handling, and AJAX
- **Improved documentation** - Clearer method documentation and better error messages
